1.4. Mac Users: Visual Studio for Mac¶

Before you start the installation guide, read through each step below. You will find the recommended features and functionality to install listed in each step.

Warning

Allow at least an hour for the installation process.

1.4.1. Installation Steps¶

Walk through the installation process by following Microsoft’s Install Visual Studio for Mac guide.

Prerequisites

Check out that you can install this on your computer.

Install Visual Studio

  1. Download the installer.

  2. Install using the .dmg file.

  3. Open the application.

  4. Continue if you agree with the privacy statements and license terms.

  5. The only component required for this course is .NET.

    You can modify this later as you gain more coding experience.

  6. Select the Install button.

  7. Assist the installer as needed.

    This may take a while, but stick around. You may be asked to enter your password a few times.

  8. To personalize or not? This is up to you.

    Microsoft Account

    Creating a Microsoft account is optional.

    Keyboard Shortcuts

    The default is Visual Studio for Mac and that is a good starting option. You can change this later.

  9. Launch Visual Studio and start developing

1.4.2. A Tour of Your New IDE¶

Microsoft has created a walkthrough of the Visual Studio 2022 for Mac. We recommend you read through this to learn the location of key components within your new IDE.

1.4.3. Troubleshooting¶

1.4.3.1. Already have Visual Studio on Your Computer?¶

If you don’t have this most recent version of Visual Studio installed, you will need to updated it. The Update Visual Studio for Mac Guide can help you update your old version of VS to the latest. At the time of this writing, the current version for the course is 2022.

Updating can also help you keep any of your workloads, extensions, or other installed packages up to date.
